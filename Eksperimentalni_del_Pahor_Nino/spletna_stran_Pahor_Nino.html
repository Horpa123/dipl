Primera CSS animacije

.kocka {
width: 200px;
height: 200px;
position: relative;
animation-name: kocka;
animation-duration: 3s;
animation-iteration-count: infinite;
animation-direction: alternate;
}

@keyframes kocka {
from {
left: -500px;
}
to {
left: 500px;
}
}
Premikajoča se kocka je bila dosežene le z uporabo CSS-ja. Kocko predstavlja DIV, kateri se po intervalih premika od točke do točke. Animiran pa je z parametri:
animation-name: kocka, s katero deklariramo ime animacije,
animation-duration: 3s s katero nastavimo čas animacije na 3 sekunde,
animation-iteration-count: infinite s katero določimo kolikokrat se animacija izvrši
animation-direction: alternate s katerim določimo smer animacije

Sledi še CSS pravilo @keyframes, s katerim omogočimo premik kocke. from določa začetno točko kvadrata, katera je na poziciji -500px, to pa določa končni položaj na desni strani z vrednostjo 500px. Vse te funkcije skupaj povzročijo, da se kvadrat premika levo in desno v neskončnost.
.krog-animacija{
width: 200px;
height: 200px;
background-color: #DB504A;
border-radius: 50%;
animation-name: krog;
animation-duration: 3s;
animation-iteration-count: infinite;
animation-timing-function: ease-in-out;
}

@keyframes krog {
0% {
transform: scale(1);
opacity: 1;
color: #DB504A;
}
50% {
transform: scale(1.3);
background-color: #E85D75;
border-radius: 0%;
}
100% {
transform: scale(1);
opacity: 1;
color: #DB504A;
}
}
Pravila animacije smo določili enako kot pri prejšnem primeru, dodana je bila le ena dodatna funkcija:
animation-timing-funciton: ease-in-out, katera povzroči, da se animacija začne počasi, pospeši na sredini in nato se nazaj upočasni. Sledi še pravilo @keyframes, ki definira našo animacijo poimenovano krog

0% določa začetni položaj kroga, kjer se uporabijo naslednje lastnosti:
transform: scale(1) določa začetno velikost kroga, ki je 1.
opacity: 1 določa začetno vidnost kroga, ki je popolnoma viden.
color: #DB504A določa začetno barvo besedila, ki je rdeča.

50% določa srednji položaj animacije, kjer se uporabijo naslednje lastnosti:
transform: scale(1.3) določa povečanje velikosti kroga na 1.3.
background-color: #E85D75 določa spremembo barve ozadja kroga v rožnato.
border-radius: 0% določa spremembo radija kroga na 0%, kar povzroči, da se krog spremeni v kvadrat.

100% določa končni položaj animacije, kjer se uporabijo naslednje lastnosti:
transform: scale(1) določa velikost kroga nazaj na 1.
opacity: 1 ohranja vidnost kroga, ki ostaja popolnoma viden.
color: #DB504A vrne barvo besedila na začetno rdečo barvo.
Uporaba Javascripta
Veliko število animacij na spletnih straneh se odziva glede na našo pozicijo na strani. Katero lahko pridobimo z javascript-ovim poslušalcem window.scrollY, kateri nam vrne našo pozicijo v pikslih na Y osi.
window.addEventListener('scroll', () =>{
document.getElementById("scroll-prikaz").innerHTML = window.scrollY;
})
Ta koda doda poslušalca dogodkov na okno brskalnika, ki posluša dogodek scroll, kar pomeni, da se koda sproži, ko uporabnik premakne drsnik na spletni strani.

Ko se dogodek scroll sproži, se sproži funkcija, ki poišče element s ID-jem scroll-prikaz (kar je id DIV-a prikaza števila) in mu nastavi vsebino na vrednost window.scrollY.

window.scrollY predstavlja število pikslov, za katere se je pomaknila spletna stran navpično od vrha okna. Ta vrednost se nato prikaže v elementu scroll-prikaz.

Z drugimi besedami, koda prikaže uporabniku, za koliko pikslov se je pomaknila spletna stran navpično, ko uporabnik premakne drsnik na spletni strani.

V tem primeru s uporabo Javascript-a glede na Y lokacijo spreminja rotacija objekta. To dosežemo s funkcijo style.transform s katero sežemo v CSS in integriramo dobljeno vrednost Y v količino rotacije kvadrata.

0 deg
const scrollDemo = document.querySelector("#primer1");

scrollDemo.addEventListener("scroll", event =>{
let image = document.getElementById("kvadrat");
image.style.transform ="rotate(" + scrollDemo.scrollTop/2 + "deg)";
document.getElementById("primer1_prikaz").innerHTML = Math.round(scrollDemo.scrollTop/2)+" deg";
}, );
Ta koda doda poslušalca dogodkov na element z ID-jem "primer1" (DIV v katerem se nahaja kvadrat), kateri posluša dogodek "scroll", kar pomeni, da se koda sproži, ko uporabnik premakne drsnik v tem elementu.

Ko se dogodek "scroll" sproži, se sproži funkcija, ki najprej poišče element s ID-jem "kvadrat" in mu nastavi CSS lastnost transform z uporabo vrednosti scrollDemo.scrollTop/2. Ta vrednost se pridobi s deljenjem trenutne pozicije drsnika v elementu z ID-jem "primer1" z 2.

scrollDemo.scrollTop predstavlja število pikslov, za katere se je pomaknila vsebina elementa navpično od vrha. S tem izračunom se slika z ID-jem "kvadrat" rotira glede na pozicijo drsnika, kar uporabniku omogoča interakcijo s spletno stranjo.

Nato se v element z ID-jem "primer1_prikaz" prikaže število stopinj, za katere se je slika z ID-jem "kvadrat" zavrtela, s pomočjo Math.round(scrollDemo.scrollTop/2)+" deg". Funkcija Math.round() zaokroži število stopinj na najbližje celo število.

Z drugimi besedami, koda uporabniku omogoča, da z drsnikom v elementu upravlja rotacijo kvadrata, pri čemer se trenutna rotacija prikaže v elementu s prikazom.
